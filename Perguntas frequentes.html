Respostas

O que é uma manifestação?

Manifestação é a forma pela qual você pode expressar suas opiniões, dúvidas, reclamações, solicitações e elogios aos serviços públicos.

Quais são os tipos de manifestação?

  • Solicitação
  • Elogio
  • Sugestão
  • Reclamação
  • Denúncia
  • Pedido de Acesso à Informação

Quem pode se manifestar?

Qualquer pessoa física ou jurídica pode se manifestar.

Como posso fazer uma manifestação?

Você pode fazer uma manifestação pela internet, presencialmente, por telefone, carta ou e-mail, dependendo das opções oferecidas.

O que é o sistema de Ouvidoria?

O sistema de Ouvidoria é uma plataforma integrada para registro e encaminhamento de manifestações aos órgãos competentes.

Como acompanhar o andamento da minha manifestação?

Você pode acompanhar pelo sistema usando seu login e senha, ou pelo número de protocolo e código de acesso gerado durante o registro.

É possível alterar minha manifestação depois que foi enviada?

Não é possível alterar a manifestação depois de enviada, mas é possível complementá-la se solicitado pela Ouvidoria.

Posso incluir anexos na manifestação?

Sim, você pode incluir até 10 anexos por manifestação, como documentos, imagens, PDFs, entre outros.

É necessário me identificar para fazer uma manifestação?

Manifestações como reclamação, solicitação, sugestão, elogio e pedido de acesso requerem cadastro. Denúncias podem ser feitas anonimamente, mas sem número de protocolo ou resposta da ouvidoria.

Quais as garantias de proteção à minha identidade?

Suas informações pessoais são protegidas pela Lei de Acesso à Informação e pela Lei Geral de Proteção de Dados (LGPD).

O que acontece com minha manifestação após o registro?

Sua manifestação é avaliada para identificar a melhor forma de tratamento: resposta, complementação, encaminhamento ou orientação.

Qual o prazo para receber a resposta?

O prazo padrão é de 30 dias, prorrogáveis por mais 30 dias mediante justificativa.

Como posso consultar o andamento da minha manifestação?

Use seu cadastro no sistema ou o número de protocolo gerado para acompanhar.

Posso alterar uma manifestação já enviada?

Não é possível editar após o envio, mas é permitido complementar se solicitado pela Ouvidoria.

Posso incluir anexos na minha manifestação?

Sim, até 10 anexos de até 30MB no total.

Posso fazer várias denúncias em uma só manifestação?

Sim, mas é recomendável separar para facilitar a apuração.

Existe proteção para servidores que fazem denúncias?

Sim, há proteção legal para servidores que denunciam irregularidades.

Onde posso encontrar dados estatísticos sobre ouvidorias?

Consulte o Painel Resolveu para estatísticas atualizadas.

Quem gere e mantém o sistema?

A administração do sistema é feita pela Ouvidoria Dineng.

Onde o sistema é hospedado?

O sistema é hospedado em servidores seguros e gerenciados pela Dineng.

O sistema tem algum custo?

O uso do sistema é gratuito, não há custos adicionais.

Como instalar o sistema?

O sistema funciona totalmente online, não requer instalação local, apenas acesso através de cadastro no sistema.

Como aderir ao sistema?

Acesse o sistema e siga os passos para criar sua conta e começar a utilizar.

Existe ambiente de treinamento para aprender a usar o sistema?

Sim, há ambiente de treinamento disponível para novos usuários.

Como obter acesso aos ambientes de treinamento e produção?

Solicite acesso pelo sistema ou entre em contato com o suporte técnico.

O que é a Ouvidoria Interna?

É a área responsável por tratar manifestações internas dos servidores da Ouvidoria Dineng.

Quais manifestações posso fazer na Ouvidoria Interna?

Manifestações relacionadas ao ambiente de trabalho, ética e conduta profissional.
